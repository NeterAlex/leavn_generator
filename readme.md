## Leaving Note Generator - LeavNG 假条生成器

### 技术栈

- 使用 `React` 和 `Vite` 构建
- 使用 `docx-templater` 实现对 word 文档模板的操作

### 文件说明

- `public/leave_note.docx` - 假条模板文件
- `src/components` - React 组件文件夹
- `src/utils` - 生成假条过程中用到的工具函数
- `src/main.tsx` - 应用程序入口点

### 调试与构建

- 运行 `npm run dev` 进行调试
- 运行 `npm run build` 构建静态文件
